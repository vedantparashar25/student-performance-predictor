# Student Performance Predictor

## 📌 Overview
This project predicts whether a student will pass or fail based on:
- Study hours
- Attendance
- Previous marks

## ⚙️ Technologies Used
- Python
- Pandas
- Scikit-learn

## 🚀 How to Run

1. Install dependencies:
pip install pandas scikit-learn

2. Train model:
python train.py

3. Run prediction:
python predict.py

## 📊 Example

Input:
Study hours: 5  
Attendance: 75  
Marks: 60  

Output:
PASS

## 🎯 Purpose
Helps students estimate performance and improve study planning.
